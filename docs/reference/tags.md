## Utilisation des Tags

Les tags permettent de trouver rapidement du contenu connexe. Cliquez sur un tag dans n'importe quelle page pour voir toutes les pages avec ce tag.

### Ajouter des Tags

Dans l'en-tête YAML d'une page :

```yaml
---
tags:
  - guide
  - installation
---
```
