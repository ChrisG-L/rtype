# Contrôles

## Clavier

| Action | Touche | Alternative |
|--------|--------|-------------|
| **Haut** | ++arrow-up++ | ++w++ |
| **Bas** | ++arrow-down++ | ++s++ |
| **Gauche** | ++arrow-left++ | ++a++ |
| **Droite** | ++arrow-right++ | ++d++ |
| **Tir** | ++space++ | ++enter++ |
| **Pause** | ++escape++ | - |
| **Voice Chat** | ++v++ | - |

### Raccourcis Système

|--------|--------|

## Rebinding (7 touches)

Personnalisez dans **Options → Contrôles** ou `config/controls.json` :

```json
{
  "keyboard": {
    "move_up": "W",
    "move_down": "S",
    "move_left": "A",
    "move_right": "D",
    "shoot": "Space",
    "pause": "Escape",
    "voice_chat": "V"
  }
}
```

## Astuces

!!! tip "Mouvement diagonal"
    Le déplacement en diagonale est **~41% plus rapide**. Utilisez-le pour esquiver !

!!! warning "Bords de l'écran"
    Évitez les coins, vous aurez moins d'options d'esquive.
