# Leaderboard

Consultez les classements, vos statistiques et vos succès.

## Accès

Cliquez sur le bouton **LEADERBOARD** dans le menu principal.

---

## Onglets

Le leaderboard comprend 3 onglets :

### Classement

Liste des meilleurs joueurs avec :

| Colonne | Description |
|---------|-------------|
| **RANK** | Position dans le classement |
| **PLAYER** | Nom du joueur |
| **SCORE** | Score total |
| **WAVE** | Vague atteinte |
| **KILLS** | Nombre d'ennemis tués |
| **TIME** | Durée de la partie |
| **MODE** | Mode de jeu (visible uniquement avec filtre "ALL") |

!!! tip "Tri par colonnes"
    Cliquez sur l'en-tête d'une colonne pour trier le classement par cette valeur.

### Statistiques

Vos statistiques personnelles :

- Parties jouées
- Kills / Morts (ratio K/D)
- Meilleur score et meilleure vague
- Temps de jeu total
- Kills par type d'arme (Standard, Spread, Laser, Missile)

### Achievements

10 succès à débloquer :

| Succès | Condition |
|--------|-----------|
| **First Blood** | Obtenir 1 kill |
| **Exterminator** | 1000 kills au total |
| **Combo Master** | Atteindre un combo x3.0 |
| **Boss Slayer** | Tuer un boss |
| **Survivor** | Atteindre la vague 20 sans mourir |
| **Speed Demon** | Vague 10 en moins de 5 minutes |
| **Perfectionist** | Compléter une vague sans prendre de dégâts |
| **Veteran** | Jouer 100 parties |
| **Untouchable** | Terminer une partie avec 0 mort |
| **Weapon Master** | 100+ kills avec chaque arme |

---

## Filtrage par Mode de Jeu

Filtrez le classement selon le nombre de joueurs :

| Filtre | Description |
|--------|-------------|
| **ALL** | Tous les modes confondus |
| **SOLO** | Parties en solo (1 joueur) |
| **DUO** | Parties à 2 joueurs |
| **TRIO** | Parties à 3 joueurs |
| **4P** | Parties à 4 joueurs |
| **5P** | Parties à 5 joueurs |
| **6P** | Parties à 6 joueurs |

!!! note "Colonne MODE"
    La colonne **MODE** n'apparaît que lorsque le filtre **ALL** est sélectionné, pour identifier le type de partie de chaque score.

---

## Filtrage par Période

| Période | Description |
|---------|-------------|
| **All-Time** | Tous les scores depuis le début |
| **Weekly** | Scores des 7 derniers jours |
| **Monthly** | Scores des 30 derniers jours |

---

## Rang en Temps Réel

Pendant la partie, votre rang global est affiché dans le HUD (coin supérieur droit).

### Indicateurs

| Élément | Description |
|---------|-------------|
| **RANK #X** | Votre position globale |
| **BEST: XXX** | Votre meilleur score personnel |

### Couleurs du rang

| Position | Couleur |
|----------|---------|
| #1 | Or |
| #2 | Argent |
| #3 | Bronze |
| Top 10 | Bleu clair |
| Top 50 | Vert |
| Autres | Gris |

!!! success "Nouveau record"
    Quand votre score actuel dépasse votre meilleur score, l'indicateur **BEST** devient vert et affiche **NEW!**

## Navigation

| Touche | Action |
|--------|--------|
| ++escape++ | Retour au menu principal |
| Clic sur onglet | Changer d'onglet |
| Clic sur filtre | Appliquer le filtre |

---

## Dépannage

??? question "Mon score n'apparaît pas"
    - Le classement se met à jour toutes les 10 secondes
    - Vérifiez que vous êtes connecté au serveur
    - Les scores sont enregistrés à la fin de la partie

??? question "Je ne vois pas mon rang pendant la partie"
    - Le rang s'affiche uniquement si vous avez déjà joué au moins une partie
    - Vérifiez votre connexion réseau
